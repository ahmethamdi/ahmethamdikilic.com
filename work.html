/01 · Shopify Apps

Custom & public Shopify app'ler.

APP / 01

Mega Menu Builder

Merchant'ların tema kodu yazmadan advanced mega menü kurabildiği custom Shopify app. Drag-and-drop builder, multi-level navigation (collection, ürün, link, visual blok), ayrı mobile/desktop config, Theme App Extensions ile high-performance render.

  • Online Store 2.0 uyumlu
  • Image, icon, kampanya block'ları
  • Mobile/desktop ayrı config
APP / 02

ETAly · Delivery Date & Shipping

Conversion-focused Shopify app: PDP, sepet, checkout ve thank-you sayfasında dinamik teslimat tarihi. Shipping rule, cut-off time, hafta sonu, tatil ve ülke-spesifik logic. Ürün/koleksiyon/tag/varyant target. Multi-language, multi-country.

  • Checkout Extensibility entegrasyonu
  • Analytics dashboard
  • Free / Pro / Advanced tier

/02 · Integrations

Marketplace & ERP senkronizasyonları.

INT / 01

Shopify ↔ Plentymarkets

Vapor Handel GmbH için Elfbargermany dropshipping integration. Laravel back-end, REST API üzerinden order fetching → transformation → Plentymarkets'a submit. Custom CMS ile data flow yönetimi.

  • End-to-end order sync
  • Security layers
  • Custom CMS dashboard
INT / 02

Shopify ↔ Shopware (5 & 6)

Bi-directional ürün, sipariş, müşteri ve stok senkronizasyonu. Order status, payment state ve fulfillment sync. Robust API mapping layer + retry mekanizması.

  • Bi-directional sync
  • Mapping layer + retry
  • Manuel operasyonu azalttı
INT / 03

Shopify ↔ Walmart Marketplace

Shopify ürünlerini Walmart'a sync. Otomatik inventory, pricing, ürün data sync. Walmart siparişleri Shopify'a import. Background jobs, batching, rate-limit handling.

  • Inventory + pricing sync
  • Order import → Shopify
  • Production stability
