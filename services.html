Hizmetler · /services

Üç katman, bir mimar.

Shopify mağazana hangi katmanda yardım gerekiyorsa: temadan app'e, app'ten marketplace entegrasyonuna kadar. Her hizmet için scope, deliverable ve tipik süreç aşağıda.

/ 01

Custom Shopify Theme Development

Figma'dan piksel-piksel custom Shopify teması. Online Store 2.0 mimarisi, modüler section'lar, metafield yönetimi, JSON template'ler. Hard-coded içerik yok — her şey admin'den editable.

SCOPE

Ne dahil?

  • Figma → Liquid (pixel-perfect)
  • Modüler section architecture
  • Metafield + dynamic source
  • JSON templates
  • Mobile-first responsive
  • Performance budget (CWV)
DELIVERABLE

Teslim

  • Production-ready Shopify theme
  • Editable sections + blocks
  • Theme docs (TR/EN)
  • QA on Lighthouse mobile
  • 1 ay post-launch destek
TIMELINE

Süre

  • Discovery: 3–5 gün
  • Build: 3–6 hafta
  • QA + launch: 1 hafta
  • Toplam: ~5–8 hafta
/ 02

Shopify App Development

Public ve custom Shopify app'leri. Theme App Extensions, Checkout Extensibility, Admin UI, billing, webhook altyapısı. Mega Menu Builder ve ETAly Delivery Date — örnek production app'lerim.

SCOPE

Ne dahil?

  • Custom & Public app architecture
  • Theme App Extensions
  • Checkout Extensibility
  • Admin UI (Polaris / custom)
  • Webhook + queue setup
  • Billing & subscription tiers
DELIVERABLE

Teslim

  • App + dashboard
  • App Store listing hazırlığı
  • Documentation
  • Analytics dashboard
  • Multi-language ready
TIMELINE

Süre

  • MVP: 4–8 hafta
  • Public launch: +2 hafta review
  • Post-launch iteration: aylık
/ 03

Marketplace Integration

Shopify ↔ Plentymarkets, Shopware 5/6, Walmart Marketplace senkronizasyonu. Ürün, stok, sipariş, fiyat, fulfillment senkronu. Rate-limit, retry ve error handling içeren prod-stable mimariler.

SCOPE

Ne dahil?

  • Bi-directional sync
  • Product · Inventory · Pricing
  • Order · Fulfillment
  • API mapping layer
  • Background jobs · Queues
  • Retry + error logging
DELIVERABLE

Teslim

  • Integration service (Laravel/Node)
  • Custom CMS / dashboard
  • Webhook orchestration
  • Monitoring + alerts
  • Runbook
TIMELINE

Süre

  • Discovery + mapping: 1–2 hafta
  • Build: 4–10 hafta
  • Staging + UAT: 1–2 hafta
