Intro

Ben Ahmet. Shopify mağazalarını hızlı, ölçülebilir ve gerçekten satan sistemlere dönüştürüyorum.

34devs altında çalışıyorum. Fokusum üç şey: tema mimarisi (Online Store 2.0, modüler sectionlar, metafield'ler), custom Shopify app'ler (Mega Menu Builder, ETAly delivery date, Shopify ↔ Plentymarkets/Shopware/Walmart) ve conversion mühendisliği (PDP, sepet, checkout extensibility, Core Web Vitals).

01 / THEME

Custom Shopify Theme

Figma'dan piksel-piksel custom theme. Online Store 2.0, sectionlar, metafield'ler, JSON template'ler. Hard-coded içerik yok — her şey admin'den yönetilebilir.

  • Liquid, HTML, CSS, JS
  • Mobile-first responsive
  • Performance budget'lı
  • Tailwind opsiyonel
02 / APPS

Shopify App Development

Public ve custom Shopify app'leri. Theme App Extensions, Checkout Extensions, Admin UI, billing, webhook altyapısı. Node + Laravel back-end deneyimi.

  • Mega Menu Builder
  • ETAly · Delivery Date
  • Checkout Extensibility
  • Subscription pricing
03 / SYNC

Marketplace Integrations

Shopify ↔ Plentymarkets, Shopware 5/6, Walmart Marketplace senkronizasyonu. Ürün, stok, sipariş ve fulfillment akışı; rate-limit ve retry stratejisi dahil.

  • REST + GraphQL API
  • Webhook + middleware
  • Background jobs
  • Error handling / retry
