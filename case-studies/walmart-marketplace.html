/ 06 · Marketplace — Inventory · Pricing · Orders

Shopify catalog, live on Walmart.

Shopify ürünlerini Walmart Marketplace'e otomatik sync, Walmart siparişlerini Shopify'a import. Background jobs, batching, rate-limit handling ile production-stable mimari.

Problem

Çoklu kanal = çoklu manuel.

Mağaza Shopify'da büyüyor ama büyük çoğunluğu hâlâ Walmart Marketplace'de değil — kanaldan kanala manuel listing imkansız (1000+ ürün × farklı kategori şemaları). Üstelik Walmart'ın API'si stricter rate limits ve farklı taxonomy taşıyor (Shopify product type ≠ Walmart category).

Solution

Background sync pipeline + batch + rate limit.

  • Catalog sync — Shopify product → Walmart feed XML, kategoriye göre attribute mapping
  • Inventory sync — Shopify location'larından stok hesap, Walmart'a push
  • Pricing sync — Shopify variant price → Walmart, currency + tax düzeltmesi
  • Order import — Walmart orders → Shopify draft order → Shopify order (status'a göre)
  • Background jobs — Laravel queue, supervisord worker'lar
  • Batching — Walmart 100 SKU/request limiti içinde optimize chunk
  • Rate limit handling — token bucket algorithm + automatic backoff
  • Error recovery — Guzzle retry middleware + dead letter queue

Stack

PHP + Laravel pipeline.

  • PHP 8 · Laravel 10
  • Guzzle HTTP client
  • Queue + supervisord workers
  • Token bucket rate limiter
  • Walmart Marketplace API
  • Shopify Admin API (GraphQL)
