/ 03 · Shopify App — Theme Extension

Mega menus without touching theme code.

Merchant'ların admin'den drag-and-drop ile karmaşık mega menü kurabildiği Shopify app. Theme App Extension üzerinden render olur — Shopify Online Store 2.0 uyumlu, kod gerektirmez, performans theme'i bozmadan korunur.

Type
Custom Shopify App
Year
2024 — now
Stack
Node · React · Liquid
Tier
Free / Pro / Advanced
Problem

Theme menus + apps + custom code = chaos.

Shopify'ın native menüsü hiyerarşik olmayan basit listeden ibaret. Mega menü için merchantlar ya theme code'una müdahale ediyor (her tema güncellemesinde sorun), ya iframe-based third-party app kullanıyor (yavaş, SEO'ya zararlı), ya da geliştirici tutuyor.

Ben merchantın yapabileceği bir araç istedim: kod yok, theme'e dokunmadan, gerçek Liquid render, mobile/desktop için ayrı yapı.

Solution

Theme App Extension + visual builder.

Admin tarafı (React) ile drag-and-drop builder, storefront tarafı (Liquid) ile native render. Backend Node + Express.

  • Multi-level navigation — collection, ürün, custom link, visual block
  • Image + icon + campaign block — her menü item'ı zengin içerik desteği
  • Mobile/desktop ayrı config — mobil'de farklı IA mümkün
  • Online Store 2.0 uyumlu — section blocks API
  • Theme App Extension — high-performance Liquid render, no iframe
  • Drag-and-drop ordering — Sortable.js + React
  • Subscription billing — Shopify Billing API ile Free/Pro/Advanced tier

Stack

Theme App Extension stack.

  • Node.js · Express backend
  • Shopify Polaris admin UI
  • React + Sortable.js builder
  • Liquid snippets (storefront)
  • Shopify GraphQL Admin API
  • Billing API (subscription tiers)

Capabilities

Native-level, zero-code.
