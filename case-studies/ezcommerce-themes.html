/ 07 · Theme Development — 14 stores

14 brands, 14 themes, one studio.

EZ Commerce US için farklı sektörlerden 14 mağaza teslim ettim. Olive oil'den hardware'a, eyewear'dan garden'a — her biri kendi marka diline sahip, ama hepsi Figma'dan pixel-perfect Liquid implementation, hepsi admin'den editable, hepsi mobile-first.

Client
EZ Commerce US
Year
2023
Stores
14
Stack
Liquid · HTML · CSS · JS
Scope

Marka × sektör çeşitliliği, tek prodüksiyon disiplini.

14 mağaza — gourmet olive oil (Garisar), tech hardware (TinyCircuits), EV charging (Primecom), eyewear (Zuna), garden (Royal House Garden), candy factory (Sarah's), green tea (IE Green Tea), printing (Pera Print), pet carrier (GoGoVie), beauty (Elyfer) ve daha fazlası. Her birinin typography, color, layout ayrı; ama temel mimari aynı:

  • Figma → Liquid pixel-perfect — spacing, typography, color tokenları birebir
  • Online Store 2.0 — JSON template'ler, section-blocks API
  • Modüler section'lar — homepage, koleksiyon, ürün, kampanya sayfası için
  • Dynamic source + metafields — hard-coded içerik yok
  • Mobile-first responsive — her layout breakpoint'te tested
  • Performance budget — LCP < 2.5s, CLS < 0.1
  • SEO foundation — semantic HTML, schema, sitemap, canonical

Approach

Per-store discipline, same core.

  • Figma source-of-truth
  • Liquid + HTML + CSS + JS
  • Section-blocks API (OS 2.0)
  • Metafields + dynamic sources
  • Lighthouse mobile pass
  • SEO + analytics ready

Outcomes

14 live stores, 14 industries.
