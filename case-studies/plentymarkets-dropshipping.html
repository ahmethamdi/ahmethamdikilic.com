/ 02 · Integration — Dropshipping Sync

Shopify ↔ Plentymarkets, fully automated.

Vapor Handel GmbH için Elfbargermany dropshipping operasyonunu otomatikleştiren Laravel-based integration. Sipariş geldikçe Plentymarkets'a iletilir, stok ve fiyat geri çekilir, custom CMS üzerinden tüm akış izlenebilir.

Client
Vapor Handel GmbH
Year
2024 — now
Stack
Laravel · REST · MySQL
Role
Senior Integration Dev
Problem

Manuel sipariş aktarımı, ölçeklenemeyen operasyon.

Elfbargermany Shopify'da satış yapıyor, ama fulfillment Plentymarkets üzerinden işliyor. Her sipariş için ürün eşleştirme + stok kontrolü + manuel data entry gerekiyordu. Günde 100+ sipariş olduğunda bu artık sürdürülemez — hatalar artıyor, müşteri geri bildirimleri düşüyor, ekip yoruluyordu.

Çözüm: otomatik bi-directional sync. Ama Shopify ve Plentymarkets'ın veri modelleri farklı — variant, location, taxation ve fulfillment alanlarında her biri kendi gerçekliğine sahip.

Solution

Laravel-based middleware + custom CMS dashboard.

Stack: Laravel · MySQL · jQuery · Postman · Git. Mimari üç katmanlı:

  • Webhook receiver — Shopify order/created webhook'unu yakalar, queue'ya yazar
  • Mapper — Shopify variant → Plenty item, address normalize, taxation logic
  • Submitter — Plentymarkets REST'e POST /orders, retry strateji ile
  • Stock puller — Plenty'den her saat stok çeker, Shopify'da günceller
  • Custom CMS — tüm akışı izleyen dashboard: bekleyen, hatalı, başarılı siparişler + tek tıkla retry
  • Security layer — webhook signature verify, IP allowlist, role-based dashboard access

Hataların kaybolmaması için her step events tablosuna log düşer. Retry mekanizması exponential backoff'la 5 deneme yapar, başarısız olursa Slack alert atar.

Stack

PHP · Laravel · MySQL

  • Laravel 10 · Queue + Horizon
  • MySQL 8 · indexed event log
  • Shopify Admin API (REST + webhooks)
  • Plentymarkets REST API
  • jQuery + Ajax for CMS dashboard
  • Postman collection for QA
