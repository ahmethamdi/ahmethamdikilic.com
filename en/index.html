Intro

I'm Ahmet. I turn Shopify stores into fast, measurable systems that actually sell.

I work under 34devs. Three focus areas: theme architecture (Online Store 2.0, modular sections, metafields), custom Shopify apps (Mega Menu Builder, ETAly, Shopify ↔ Plentymarkets/Shopware/Walmart), and conversion engineering (PDP, cart, checkout extensibility, Core Web Vitals).

Services / 03

Whichever layer your store needs.

01 / THEME

Custom Shopify Theme

Pixel-perfect custom themes from Figma. Online Store 2.0, sections, metafields, JSON templates. Nothing hard-coded — everything is admin-editable.

  • Liquid, HTML, CSS, JS
  • Mobile-first responsive
  • Performance budget
  • Tailwind optional
02 / APPS

Shopify App Development

Public and custom Shopify apps. Theme App Extensions, Checkout Extensions, Admin UI, billing and webhook infrastructure. Node + Laravel back-end.

  • Mega Menu Builder
  • ETAly · Delivery Date
  • Checkout Extensibility
  • Subscription pricing
03 / SYNC

Marketplace Integrations

Shopify ↔ Plentymarkets, Shopware 5/6, Walmart Marketplace sync. Product, inventory, order and fulfillment flow with rate-limit and retry strategy.

  • REST + GraphQL
  • Webhook + middleware
  • Background jobs
  • Error handling / retry
