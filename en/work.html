/01 · Shopify Apps

Custom & public Shopify apps.

APP / 01

Mega Menu Builder

Custom Shopify app letting merchants build advanced mega menus without touching theme code. Drag-and-drop builder, multi-level navigation (collection, product, link, visual block), separate mobile/desktop config, high-performance render via Theme App Extensions.

  • Online Store 2.0 compatible
  • Image, icon, campaign blocks
  • Separate mobile/desktop config
APP / 02

ETAly · Delivery Date & Shipping

Conversion-focused Shopify app: dynamic delivery date on PDP, cart, checkout and thank-you page. Shipping rule, cut-off time, weekend, holiday and country-specific logic. Product/collection/tag/variant targeting. Multi-language, multi-country.

  • Checkout Extensibility integration
  • Analytics dashboard
  • Free / Pro / Advanced tier

/02 · Integrations

Marketplace & ERP sync.

Shopify ↔ Plentymarkets

Elfbargermany dropshipping integration for Vapor Handel GmbH. Laravel back-end, REST API order fetching → transformation → submission to Plentymarkets. Custom CMS for data flow management.

Bi-directional product, order, customer and inventory sync. Order status, payment state and fulfillment sync. Robust API mapping layer + retry mechanism.

Sync Shopify products to Walmart. Automated inventory, pricing, product data sync. Walmart orders imported into Shopify. Background jobs, batching, rate-limit handling.
