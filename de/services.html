Drei Schichten, ein Architekt.

Welche Schicht dein Shopify Store braucht: vom Theme zur App, von der App zur Marketplace-Integration. Scope, Deliverable und typischer Zeitrahmen unten.

/ 01

Custom Shopify Theme Development

Pixel-perfektes Custom Shopify Theme aus Figma. Online Store 2.0 Architektur, modulare Sections, Metafield Management, JSON Templates. Nichts hard-coded — alles im Admin editierbar.

SCOPE

Was ist enthalten

  • Figma → Liquid (pixel-perfect)
  • Modulare Section Architektur
  • Metafield + Dynamic Source
  • JSON Templates
  • Mobile-first responsive
  • Performance Budget (CWV)
DELIVERABLE

Lieferung

  • Production-ready Shopify Theme
  • Editierbare Sections + Blocks
  • Theme Docs (TR/EN)
  • QA auf Lighthouse Mobile
  • 1 Monat Post-Launch Support
TIMELINE

Dauer

  • Discovery: 3–5 Tage
  • Build: 3–6 Wochen
  • QA + Launch: 1 Woche
  • Total: ~5–8 Wochen
/ 02

Shopify App Development

Public und Custom Shopify Apps. Theme App Extensions, Checkout Extensibility, Admin UI, Billing, Webhook-Infrastruktur. Mega Menu Builder und ETAly Lieferdatum — Beispiel-Production-Apps.

SCOPE

Was ist enthalten

  • Custom & Public App Architektur
  • Theme App Extensions
  • Checkout Extensibility
  • Admin UI (Polaris / custom)
  • Webhook + Queue Setup
  • Billing & Subscription Tiers
DELIVERABLE

Lieferung

  • App + Dashboard
  • App Store Listing Vorbereitung
  • Documentation
  • Analytics Dashboard
  • Multi-language ready
TIMELINE

Dauer

  • MVP: 4–8 Wochen
  • Public Launch: +2 Wochen Review
  • Post-Launch Iteration: monatlich
/ 03

Marketplace Integration

Shopify ↔ Plentymarkets, Shopware 5/6, Walmart Marketplace Synchronisation. Produkt, Lager, Bestellung, Preis, Fulfillment Sync. Production-stabile Architekturen mit Rate-Limit, Retry und Error Handling.

SCOPE

Was ist enthalten

  • Bi-direktionale Sync
  • Produkt · Bestand · Preise
  • Bestellung · Fulfillment
  • API Mapping Layer
  • Background Jobs · Queues
  • Retry + Error Logging
DELIVERABLE

Lieferung

  • Integration Service (Laravel/Node)
  • Custom CMS / Dashboard
  • Webhook Orchestration
  • Monitoring + Alerts
  • Runbook
TIMELINE

Dauer

  • Discovery + Mapping: 1–2 Wochen
  • Build: 4–10 Wochen
  • Staging + UAT: 1–2 Wochen
